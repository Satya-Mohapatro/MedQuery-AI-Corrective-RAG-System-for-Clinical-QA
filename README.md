# MedQuery-AI-Corrective-RAG-System-for-Clinical-QA
An intelligent medical question-answering system built on  a LangGraph-orchestrated Corrective RAG pipeline, combining  ChromaDB vector retrieval with Gemini LLM to deliver  grounded, hallucination-reduced responses from clinical  documents — evaluated against Naive RAG across Recall@5,  MRR, Faithfulness, and Hallucination Rate metrics.
